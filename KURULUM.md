# Botanik Medulla Reçete Takip Botu


## Kurulum

1. Python'un yüklü olduğundan emin olun (Python 3.7+) Yoksa indirip kurun.

Cmd yi çalıştırıp aşağıdaki kodu çalıştırın
2. Gerekli kütüphaneleri yükleyin:
```bash
pip install -r requirements.txt
```

## Kullanım

1. Medulla programını açın
2. Reçete ekranına gelin
3. Botu çalıştırın:

```bash
python botanik_bot.py
```

## Özellikler

- [x] Medulla programına otomatik bağlanma
- [x] İlaç butonuna otomatik tıklama
- [ ] Reçete bilgilerini okuma (yakında)
- [ ] Otomatik reçete girişi (yakında)
- [ ] Excel'den toplu reçete yükleme (yakında)
