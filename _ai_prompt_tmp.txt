# SUT Kuralı Implementasyon Talimatı

## PROJE: BotanikTakip - Reçete Rapor Kontrol Modülü

### SİSTEM TANIMI
Bu sistem Türkiye'deki SGK Medula eczane sistemindeki reçeteleri otomatik kontrol eder.
Her ilacın SUT (Sağlık Uygulama Tebliği) kuralına göre uygunluğunu algoritmik olarak denetler.

### GÖREV
Aşağıdaki ilaç için SUT kuralı henüz implemente edilmemiş. Yapman gereken:
1. Bu ilacın SUT kuralını internetten araştır (ilacabak.com, SGK SUT tam metni, eczacı odası kaynakları)
2. Algoritmik kontrol kodunu `recete_kontrol/sut_kontrolleri.py` dosyasına ekle
3. Test et

### İLAÇ BİLGİLERİ
- İlaç Adı: XARELTO 20 MG 28 FTB
- Etkin Madde: Rivaroksaban 15-20 mg Ağızdan katı
- Rapor Kodu: 04.03
- Rapor/Mesaj Metni: Yok

#### Adım 4: Rapor Kodu Eşleştirmesi (gerekirse)

#### Adım 5: Kontrol Fonksiyonu Yaz (yeni kategori ise)
```python
def kontrol_yeni_kategori(ilac_sonuc: Dict) -> KontrolRaporu:
    metin = _tum_metinleri_birlesir(ilac_sonuc)
    if not metin:
        return KontrolRaporu(KontrolSonucu.KONTROL_EDILEMEDI, "Rapor/mesaj metni yok")
    
    # Türkçe güvenli arama kullan
    ibare_var = _turkce_ara(metin, 'aranan ibare')
    
    if ibare_var:
        return KontrolRaporu(KontrolSonucu.UYGUN, "SUT ibaresi raporda mevcut")
    return KontrolRaporu(KontrolSonucu.KONTROL_EDILEMEDI, "SUT ibaresi bulunamadı")
```

#### Adım 6: Sözlüklere Kaydet
`KATEGORI_KONTROL_FONKSIYONU` ve `KATEGORI_ISIMLERI` sözlüklerine ekle.

### MEVCUT YARDIMCI FONKSİYONLAR

```python
# Türkçe karakter farkı gözetmeden arama (İ/i/ı, Ü/ü, Ö/ö, Ş/ş, Ç/ç, Ğ/ğ hepsi aynı)
_turkce_ara(metin, aranan) → bool

# Tüm metin kaynaklarını birleştir (mesaj + rapor açıklamaları + tanılar)
_tum_metinleri_birlesir(ilac_sonuc) → str

# Kontrol sonucu döndür
KontrolRaporu(sonuc=KontrolSonucu.UYGUN, mesaj="...", detaylar={}, uyari="...")
KontrolSonucu.UYGUN / UYGUN_DEGIL / KONTROL_EDILEMEDI / ATLANDI
```

### MEVCUT KATEGORİLER (referans)
- DIYABET_DPP4_SGLT2: Diyabet ilaçları (metformin, sülfonilüre, SGLT2, DPP4, insülin)
- STATIN: Lipid düşürücüler (atorvastatin, rosuvastatin - LDL kontrolü)
- FIBRAT: Trigliserid düşürücüler (fenofibrat)
- KLOPIDOGREL: Antiplateletler (klopidogrel, prasugrel, tikagrelor)
- KOMBINE_ANTIHIPERTANSIF: Kombine tansiyon ilaçları (monoterapi ibaresi)
- IVABRADIN: İvabradin/Ranolazin/Eplerenon (NYHA, sinüs ritmi, angina)
- YOAK: Yeni oral antikoagülanlar (varfarin, INR kontrolü)
- PSIKIYATRI: Antidepresanlar/antipsikotikler (uzman raporu)
- SOLUNUM: LABA+ICS, LABA+LAMA+ICS (KOAH, astım)
- RECETE_TURU_KIRMIZI: Kırmızı reçete gerektiren (tramadol)
- RECETE_TURU_YESIL: Yeşil reçete gerektiren (biperiden)
- TRIMETAZIDIN: Rapor zorunlu kardiyovasküler
- DMAH: Düşük molekül ağırlıklı heparin (rapor zorunlu)
- RAPORSUZ_BILGILENDIRME: Raporsuz verilebilen, mesaj bilgilendirme amaçlı
- MONO_ANTIHIPERTANSIF: Tek başına tansiyon ilacı (raporsuz verilebilir)

### KIRMIZI ÇİZGİLER - KESİNLİKLE YAPMA

1. **Medula verileri**: Reçete/rapor verisi değiştirme, ekleme, silme, güncelleme KESİNLİKLE YASAK

3. **GUI kutucukları**: Hiçbir sayıyı, metni değiştirme
4. **Koordinat tıklama**: pyautogui.click(x,y) YASAK
5. **Kaydetme butonları**: E-Reçete Kaydet, İlaç Ekle, İlaç Sil butonlarına ASLA basma
